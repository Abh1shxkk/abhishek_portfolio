# Abhishek Chauhan — Portfolio

A modern, animated portfolio website built with **React + Vite + TypeScript + Tailwind CSS**.

## Features

- ✨ Beautiful UI with animations and dark/light theme
- 💼 Experience, Projects, Skills & Education sections
- 🤖 AI Chat powered by Google Gemini
- ⚡ Fully static — no backend required
- 🚀 Deployed on Vercel

## Getting Started

```bash
cd portfolio-frontend
npm install
npm run dev
```

Create `portfolio-frontend/.env.local` from `portfolio-frontend/.env.example` before starting the app, then set `GEMINI_API_KEY` in that file.

Open [http://localhost:3000](http://localhost:3000)

## Deployment

This project is configured for **Vercel**:

1. Push to GitHub
2. Import project in [vercel.com](https://vercel.com)
3. Set root directory to `portfolio-frontend`
4. Deploy!

## Tech Stack

React • TypeScript • Vite • Tailwind CSS • Framer Motion • Google Gemini AI
